필터사용 예제

currency 필터 사용

원래 : {{cValue}}

필터적용 : {{cValue | currency}}

date 필터 사용

원래 : {{dNow}}

필터적용 : {{dNow | date : 'yyyy년 mm월 dd일'}}

JSON 필터 사용

원래 : {{jObj}}

필터적용 : {{jObj | json}}

lowercase 필터 사용

원래 : {{lString}}

필터적용 : {{lString | lowercase}}

uppercase 필터 사용

원래 : {{lString}}

필터적용 : {{lString | uppercase}}

number 필터 사용

원래 : {{nValue}}

필터적용 : {{nValue | number:5}}
